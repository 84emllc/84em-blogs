# 84EM.DEV

Web development blog built with Hugo.

## Build Commands

- `hugo server -D` - Development server with drafts
- `hugo` - Production build

## Content Structure

- `content/posts/` - Blog posts (use page bundles for images)
- `content/about.md` - About page
- `content/legal/` - Legal pages (privacy policy, etc.)

## Key Files

- `hugo.toml` - Site configuration
- `assets/css/main.css` - All styles (single file)
- `layouts/` - Hugo templates
- `static/` - Static assets (fonts, icons, images)
